# Yuan 词法分析器测试用例

这个目录包含了用于测试 Yuan 词法分析器的各种 Yuan 源码文件。

## 测试文件说明

### 正常情况测试

- **keywords.yu** - 测试所有 Yuan 关键字的识别
- **identifiers.yu** - 测试各种标识符格式，包括 Unicode 支持
- **integers.yu** - 测试整数字面量（十进制、十六进制、八进制、二进制）
- **floats.yu** - 测试浮点数字面量和科学计数法
- **strings.yu** - 测试字符串和字符字面量，包括转义序列
- **operators.yu** - 测试所有运算符和标点符号
- **comments.yu** - 测试各种注释格式

### 错误情况测试

`errors/` 目录包含各种错误情况的测试：

- **invalid_characters.yu** - 无效字符和不支持的符号
- **unterminated_strings.yu** - 未终止的字符串和字符字面量
- **invalid_numbers.yu** - 无效的数字字面量格式
- **invalid_escapes.yu** - 无效的转义序列
- **unterminated_comments.yu** - 未终止的块注释

## 使用方法

这些测试文件可以用于：

1. **单元测试** - 在 C++ 单元测试中加载这些文件，验证词法分析器的输出
2. **集成测试** - 作为完整编译流程的输入文件
3. **回归测试** - 确保词法分析器的修改不会破坏现有功能
4. **性能测试** - 测试词法分析器处理各种输入的性能

## 测试覆盖范围

这些测试文件覆盖了 Yuan 语言规范中的所有词法元素：

- ✅ 所有关键字
- ✅ 标识符（包括 Unicode）
- ✅ 数字字面量（所有进制和类型后缀）
- ✅ 浮点数字面量（包括科学计数法）
- ✅ 字符串字面量（普通、原始、多行）
- ✅ 字符字面量
- ✅ 所有运算符和标点符号
- ✅ 注释（单行、块注释、文档注释）
- ✅ 错误情况处理

## 注意事项

- 错误测试文件中的语法错误是故意的，用于测试错误处理
- 某些文件可能包含未终止的字符串或注释，这是为了测试错误恢复
- Unicode 测试确保词法分析器正确处理国际化字符
